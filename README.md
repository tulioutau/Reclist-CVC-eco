# Reclist-CVC-eco
Reclist CVC para ptbr baseada nas reclist CVVC e VCV japonesas e no CVC-MINI-BRAPA. Feita para economizar alias e manter as qualidades de outros CVC ptbr.

## Informações Básicas:

- Conotação BRAPA, com extensão LATANI(não lançada, não afilidada ao team BRAPA).
- Reclist possui 57 gravações, 116 com as extras.
- Tem 3 arquivos de oto.ini: 
    1.  oto.ini --> 638 alias, todos os alias intencionais para o voicebank.
    2. oto-lite.ini --> 547 alias, o mínimo para o voicebank
    3. oto-extra.ini --> 501 alias, apenas a parte extra da reclist.
- Guide bgm: [ガイドBGM　7モーラ](https://bowlroll.net/file/259669) 100 bpm.
- Ainda não tem phonemizer própio, porém funciona com o [phonemizer padrão do OpenUtau](https://github.com/openutau/OpenUtau/wiki/Phonemizers#pt-br-cvc-brazilian-portuguese-cvc).


## Como funciona??
A reclist "economiza" o número de alias fazendo com que todos os alias VC das vogais nasais [an, en, in, on, un] sejam reduzidos para [N C] (sendo C uma consoante qualquer)

- exemplo com a palavra "antes":
  - Brapa CVC: [- an][an ch][ch i][i s-]
  - CVC eco:   [- an][N ch][ch i][i s-]

## Extensões
| Fonema | Descrição |
|----------|--------|
| N | consoante nasal adaptada à localização de onde está, parecido com o ん. |
| Ao, Ae, Ei, Ou, Ui | ditongos nasais, sendo eles respectivamentes: [an w], [an y], [en y], [on w], [un y]. |
| S | regionalização feita para manter o sotaque do gravador baseado no final de s (flore**s**, ), é permitida variação de acordo com o sotaque do falante. |
| IN, OUT | Inspiração (IN) e expiração (OUT). |
